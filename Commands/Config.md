# git config

When first setting up git you will need to work with the git config commands in other to set up your identity.

For example, your identity can be set up with the commands such as:

git config --global user.name "Rinata White"
git config --global user.email "whiteri@uindy.edu



for more infomation, view the [first time git setup] (https://git-scm.com/book/en/v2/getting-started-first-time-git-setup) documentation.
